JRecord OtherSource code directory

This directory holds the Example and Utility programs for the JRecord Project, the directories are:

  • cb2xml - cb2xml project, Cobol Copybook analysis. This package is also used in some commercial products from the likes of IBM, Computer Associates etc.
  • JRecord_Examples1 - Examples of using Clasic-JRecord.
  • JRecord_CodeGen - Program to generate JRecord-Java interface code from a Cobol Copybook. This is still a work in progress
