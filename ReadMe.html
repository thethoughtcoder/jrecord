JRecord Source code directory

This directory holds the Java source code for the JRecord Project, the directories are:

  • JRecord_Common - parts of JRecord that are shared with the RecordEditor project.
  • JRecord - The rest of JRecord.
  • OtherSource This sub directory contains the following sub projects:
    • cb2xml - cb2xml project, Cobol Copybook analysis. This package is also used in some commercial products from the likes of IBM, Computer Associates etc.
    • JRecord_CodeGen - Program to generate JRecord-Java interface code from a Cobol Copybook. This is still a work in progress
    • JRecord_Examples1 - Examples of using Clasic-JRecord.
  • JRecord_Cbl2Xml - A simple Cobol-data-file <==> Xml-file conversion utility. It is a replacement for the Dat2Xml and Xml2Dat programs in cb2xml.
  • JRecord_IO_Builder_Examples - Examples of using JRecord-IOBuilders.
